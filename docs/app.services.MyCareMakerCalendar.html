getEvents (callback, eventType)

This calendar plugin does not include a method to get events. The goal of this method is to list the events registered for the current day. To achieve such functionality the findEvent method is utilised. The startDate was set as 00:00:00 which corresponds to the fist hour, minute, second of the day and then the endDate was set to 23:59:59 which corresponds to the last hour, minute and second of the day respectively. The rest of the field values are left blank to ensure all the events within the timeframe are included. The events are then passed in an array. The method also checks if the event type is valid.
