MyCareMakerNotifications source

This is the MyCareMakerNotifications it provides a simplified wrapper class to the ngCordova $cordovaLocalNotification. As the $cordovaLocalNotfication requires a ID number the MyCareMakerNotifications object automatically increments it every time one it created. This allows the various controllers and objects using notifications to never conflict with one another.
